scsr-build-chunked-jobscripts

This script accepts a scsr-calc command and generates a command to run each chunk. Slurm jobscripts can also be created for each chunk using -J.

usage: scsr-build-chunked-jobscripts [-h] [-J] [args for scsr-calc]

Named Arguments

-J, --create-job-script

Default: False
