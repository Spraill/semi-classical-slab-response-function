scsr-cleanup-results

usage: scsr-cleanup-results [-h] [-a] [-f] [-j] [-p] [-i] [-F] [-d DIR]

Named Arguments

-a, --all

Cleanup all results (figs, jobs, pickles).

Default: False

-f, --figs

Cleanup figs.

Default: False

-j, --jobs

Cleanup jobs.

Default: False

-p, --pickles

Cleanup pickles.

Default: False

-i, --interactive

Delete results 1-by-1 after confirmation.

Default: False

-F, --force

Skip final confirmation before deletion.

Default: False

-d, --dir

Dir to cleanup (default is current directory).

Default: “/home/runner/work/semi-classical-slab-response-function/semi-classical-slab-response-function”
